# 移动客户端功能说明

## 新增功能

### 1. 自动获取本机手机号
- 打开 APP 时自动尝试读取 SIM 卡手机号
- 自动填充到登录框
- 显示绿色勾选标记表示已自动获取
- 如果无法获取（部分设备不支持），可手动输入

### 2. 通讯录集成
- 自动读取手机通讯录
- 在线用户列表显示通讯录中的名字而非手机号
- 聊天界面显示发送者的通讯录名字
- 通讯录中的联系人显示绿色头像和勾选标记
- 顶部按钮可手动刷新通讯录

### 3. 自动重连
- 网络断开或待机后自动重连
- 显示"正在重新连接..."状态
- 每 3 秒尝试重连一次
- 连接成功后自动恢复登录状态

### 4. 保存登录状态
- 首次登录后自动保存手机号
- 下次打开 APP 自动登录
- 无需重复输入手机号

## 权限说明

APP 需要以下权限：

| 权限 | 用途 |
|------|------|
| **电话** | 读取本机手机号 |
| **通讯录** | 获取联系人姓名 |
| **网络** | 连接服务器 |

首次使用时会弹出权限申请，请点击"允许"。

## 界面说明

### 登录页面
- 自动显示本机手机号（如果支持）
- 显示"已自动获取本机号码"提示
- 手动输入框用于不支持自动获取的情况

### 首页（在线用户列表）
- 群聊入口
- 在线用户列表：
  - 通讯录中的联系人：显示名字 + 绿色头像 + 勾选标记
  - 不在通讯录中：显示手机号 + 灰色头像
  - 点击可进入私聊

### 聊天页面
- 标题栏显示对方名字（通讯录中）或手机号
- 消息气泡显示发送者名字
- 通讯录中的联系人消息显示绿色名字

## 注意事项

1. **自动获取手机号**
   - 部分手机或运营商不支持自动获取
   - 双卡手机可能只能获取默认 SIM 卡号码
   - 如无法获取，请手动输入

2. **通讯录权限**
   - 必须允许通讯录权限才能显示名字
   - 拒绝权限将只显示手机号
   - 可随时在系统设置中开启权限

3. **刷新通讯录**
   - 添加新联系人后点击首页右上角刷新按钮
   - 或重新打开 APP
